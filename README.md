# Angular Art Portfolio Completo

Portafolio artístico moderno, elegante y oscuro con morados profundos.

## Cómo probarlo

```bash
npm install
npm start
```

Abre:

```text
http://localhost:4200
```

## Dónde cambiar tus obras

Edita este archivo:

```text
src/app/data/works.ts
```

## Dónde cambiar colores

Edita:

```text
src/styles.scss
```

## Publicar en Netlify

Build command:

```bash
npm run build
```

Publish directory:

```text
dist/angular-art-portfolio-completo/browser
```
